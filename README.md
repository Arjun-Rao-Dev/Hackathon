# Hackathon

## Caregiver Tracker

This repo contains two versions of the Caregiver Task Tracker:

- Web app: `index.html`, `style.css`, `script.js`, deployed with GitHub Pages.
- iOS app: `CaregiverTrackerApp.xcodeproj`, a native SwiftUI app for Xcode.

The iOS app supports caregiver task entry, task type/date/time, native local notifications, local persistence, and English, Simplified Chinese, Spanish, and Hindi UI text.

To open the iOS app:

1. Open `CaregiverTrackerApp.xcodeproj` in Xcode.
2. Select the `CaregiverTracker` target.
3. Set your Apple Developer Team in Signing & Capabilities.
4. Build/run on a simulator or iPhone.

See `APP_STORE.md` for the remaining App Store publishing steps.

Helping Elderly & Disabled People — Project Ideas
1. Simple Communication Board
A large-button, easy-to-tap web app for people who have trouble speaking (stroke survivors, autism, etc.) where they can tap pictures/icons to communicate basic needs — "I'm hungry," "I need help," "I'm in pain." No login, works on any device, immediately usable.
2. Caregiver Task Tracker
A simple website where family caregivers can log daily tasks for an elderly relative (medications given, meals eaten, doctor appointments). Helps families coordinate care across multiple people. Super practical for real families.
3. Large-Print Emergency Info Card Generator
A web tool where you enter someone's medical info (allergies, conditions, emergency contacts) and it generates a clean, large-print PDF card they can keep in their wallet or on the fridge. Huge value for elderly people living alone.
4. Brain Exercise Game for Seniors
A simple, large-text memory or word game designed specifically for elderly users — big buttons, high contrast, slow pace, no confusing menus. Helps with cognitive stimulation and is genuinely fun.
5. "Check In" Button App
A one-button website an elderly person can bookmark — they press it once a day to signal they're okay. If they don't press it, a family member gets a simple alert. Solves a real safety problem for seniors living alone.
